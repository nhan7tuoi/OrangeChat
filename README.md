# REALTIME WEB-APP CHAT APPLICATION
###### github-app:https://github.com/nhan7tuoi/OrangeChatApp
###### github-web:https://github.com/nhan7tuoi/OrangeChatWeb
###### github-server:https://github.com/NhatSang/OrangeC

##### The project uses Nodejs to build a server using mongodb to store data and EC2 to store images
##### The app client is built using react native (redux,axios,formik,socket)
##### Web client is built using reactjs
